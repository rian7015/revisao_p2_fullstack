Posicionamento CSS

Box Model

Conteúdo — padding — border — margin 
.elemento {
  width: 200px;
  height: 100px;
  padding: 20px;         /* espaço INTERNO */
  margin: 30px;          /* espaço EXTERNO */
  border: 3px solid red; /* borda */
  border-radius: 10px;   /* cantos arredondados */
  box-shadow: 5px 5px 10px gray;
}

Display: block vs inline

Sou um div — display: block (ocupa a linha toda)
Segundo div — também ocupa linha inteira
span inline outro span inline lado a lado! 
/* block: div, p, h1-h6, form, header, footer, section */
display: block;

/* inline: span, a, img */
display: inline;

/* inline-block: comporta como inline mas aceita width/height */
display: inline-block;

/* none: esconde o elemento */
display: none;

Posicionamento: static (padrão)

Elementos estáticos ignoram top, left, right, bottom.

static — fluxo normal do documento

Posicionamento: relative

Relativo à minha posição original (deslocado 20px para baixo e direita) 
.relative {
  position: relative;
  top: 20px;
  left: 20px;
}

Posicionamento: absolute

Container PAI (position: relative)
Absoluto: canto superior direito do pai 
.pai    { position: relative; }
.filho  { position: absolute; top: 0; right: 0; }

z-index — sobreposição

z-index: 1 (fundo)
z-index: 10 (frente)
.fundo  { position: absolute; z-index: 1; }
.frente { position: absolute; z-index: 10; }
